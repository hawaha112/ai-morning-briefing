Coding 留存率取代 AGI 叙事成估值锚，OpenAI 陷入叙事危机

Anthropic 本轮 9650 亿估值，投资人公开锚定的是 Claude Code 付费续约率——可计量现金流，不是通用智能想象。同期 Lovable ARR 超 4 亿美元、估值较年初翻倍到 120 亿，逻辑相同：市场在用脚投票，能拿出「用户持续付钱」硬数据的 Coding 赛道拿到溢价。OpenAI 处境最尴尬：估值叙事依赖 AGI 想象，而 Claude Code 的留存数字第一次给「谁更值钱」提供了可横向对标的硬参数。OpenAI 要守住估值，必须在 Coding 产品线拿出同级留存证明；拿不出，差距只会扩大。

Mythos Preview 52 倍代码加速是 AI 递归自改飞轮首次被内部数据计量的公开宣告

今天三件事同时落地，AI行业的估值逻辑，悄悄变了。

先说Anthropic，这轮融资估值9650亿，但投资人盯的不是"通用智能"这个大故事，盯的是Claude Code的付费续约率——就是用户用了下个月还掏不掏钱。同期一个叫Lovable的Coding产品ARR超了4亿美元、估值今年翻倍，逻辑完全一样。市场在投票：谁能拿出"用户持续付钱"的硬数据，谁就拿到溢价。这对OpenAI很不妙，它的估值还在靠AGI想象撑着，现在有了可以横向对比的留存数字，拿不出同等证明，差距只会越拉越大。

第二件事，白宫正在谈持股OpenAI，同一周特朗普签令让AI进入军事情报领域。你把这俩叠在一起看——政府一旦入股就是受益方，天然有动机把国防采购往OpenAI导；军事令再把这块需求直接做大。这不是政策倾斜，是产权绑定，Anthropic和谷歌在这个结构面前根本没切入点。

第三件，新模型Mythos Preview代码优化跑出了52倍加速。对比一下，去年Claude Opus是3倍，人类顶级工程师上限才4到8倍。Anthropic说这是"递归自改早期信号"——AI提速研究员工作，产出更强模型，更强模型再加速研究，这个飞轮第一次被内部数据量化了，不再是理论推演。

三条线同一天收紧：钱认可了可计量的东西，政府用产权下场，飞轮开始能测了。AI行业，正式从讲故事，进入交卷阶段。
